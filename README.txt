# 躲鬼遊戲

這是一個文字遊戲，玩家身處一個黑暗空間，需要避免被隱藏的鬼抓到。遊戲中，玩家將選擇位置來查看附近的鬼數量，並在不被抓到的情況下躲避所有鬼。

## 遊戲流程

1. **設定遊戲參數**：
   - 輸入空間大小 (格式：`MN`) 以及鬼的數量。
   - 範例：`5 5` 表示空間為 5x5。

2. **開始遊戲**：
   - 玩家可以選擇一個位置來查看該位置的狀態。
   - 如果該位置有鬼，顯示 `X`，並立即結束遊戲。
   - 若無鬼，顯示周圍 8 格的鬼的數量。

3. **結束條件**：
   - 玩家被鬼抓到，遊戲結束並顯示「你被鬼抓到了」。
   - 若玩家成功查看完所有非鬼的位置，顯示「你成功躲避所有的鬼了」。

## 錯誤處理

- 如果空間大小小於或等於鬼的數量，顯示「遊戲參數錯誤」，並結束遊戲。
- 輸入的查看位置超出範圍或重複，顯示「無效輸入，請再試一次」。
